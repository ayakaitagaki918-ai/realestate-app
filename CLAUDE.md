# プロジェクト概要
不動産アプリ（realestate-app）のWebアプリです。

## 技術スタック
- HTML / CSS / JavaScript

## コーディング規約
- コメントは日本語で書く
- 変数名は英語のキャメルケース（例：userName）を使う

## 禁止事項
- `rm -rf` コマンドは絶対に実行しない
- `.env` ファイルを読んだり変更したりしない
- `package.json` の依存パッケージを無断で変更しない
- データベースへの削除操作（DELETE文）を実行しない

## 返答ルール
- 返答は必ず日本語で行う
